HTML adalah bahasa untuk menyusun dan menampilkan konten web menggunakan tag, bekerja bersama CSS dan JavaScript untuk interaktivitas.

CSS

CSS (Cascading Style Sheets) adalah bahasa untuk mengatur tampilan dan desain elemen-elemen HTML di halaman web, seperti warna, layout, dan font

JavaScript adalah bahasa pemrograman untuk menambahkan interaktivitas, animasi, dan fungsionalitas dinamis pada halaman web di browser.
